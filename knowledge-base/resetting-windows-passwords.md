# Resetting Windows Passwords

## Purpose

This article provides a standard procedure for resetting a user's Windows password and verifying successful account access.

---

## Symptoms

Users may report:

* Unable to log into Windows
* Incorrect password message
* Forgotten password
* Account locked after multiple failed attempts

---

## Troubleshooting Steps

1. Verify the user's identity according to company policy.
2. Open Active Directory Users and Computers.
3. Locate the user account.
4. Reset the user's password.
5. Select **User must change password at next logon** if required.
6. Unlock the account if it is locked.
7. Confirm the account is enabled.
8. Provide the temporary password securely.
9. Ask the user to sign in and create a new password.

---

## Resolution

The user's password was successfully reset, account access was restored, and the user successfully authenticated using the temporary password.

---

## Notes

Never communicate passwords through unsecured channels. Always verify user identity before performing password resets.
