# Project Overview

## Purpose

The goal of this project was to build and configure a functional osTicket help desk environment that simulates the daily responsibilities of an entry-level IT Support Technician.

The environment was designed to replicate common help desk operations, including ticket management, technician assignments, customer communication, troubleshooting documentation, and ticket resolution.

---

## Objectives

- Deploy a working osTicket environment using Docker
- Configure a realistic help desk structure
- Create departments and technician accounts
- Simulate end-user support requests
- Document troubleshooting using technician notes
- Practice professional customer communication
- Complete realistic support tickets from creation through closure

## Project Scope

This project simulates a small business IT Help Desk supporting users experiencing common technical issues.

Five realistic support scenarios were completed covering:

- Windows password reset
- Office printer troubleshooting
- Network drive mapping
- Microsoft Outlook synchronization
- Network connectivity troubleshooting

Each ticket demonstrates the complete support workflow from initial user request through successful resolution.
