# Ticket Scenarios

This project includes five simulated IT support tickets that demonstrate common Help Desk responsibilities.

---

## Ticket 1 — Windows Password Reset

**Issue**

User unable to log into Windows.

**Resolution**

Verified account status, reset password, documented the resolution, and confirmed successful access.

---

## Ticket 2 — Office Printer Issue

**Issue**

Printer stopped processing print jobs.

**Resolution**

Verified printer connectivity, cleared the print queue, restarted the print spooler service, and confirmed successful printing.

---

## Ticket 3 — Network Drive Mapping

**Issue**

Mapped S: drive unavailable after login.

**Resolution**

Verified connectivity, confirmed file server access, remapped the network drive, and validated user access.

---

## Ticket 4 — Outlook Synchronization

**Issue**

Microsoft Outlook unable to send or receive email.

**Resolution**

Verified Microsoft 365 connectivity, repaired the Outlook profile, synchronized the mailbox, and confirmed successful email delivery.

---

## Ticket 5 — Network Connectivity

**Issue**

No internet connectivity after workstation relocation.

**Resolution**

Verified physical connectivity, renewed the DHCP lease, validated DNS and gateway configuration, tested connectivity, and restored network access.
